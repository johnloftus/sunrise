# sunrise

Sunrise is a browser-based tool to calculate and display the grey-line (terminator) on a Mercator map.  This tool is designed for radio propagation analysis, particularly for understanding grey-line opportunities between two locations.

## 🌍 Features

- Displays short-path and long-path between two cities
- Calculates and renders the grey-line (terminator)
- Supports multiple twilight definitions:
  - Civil (-6°)
  - Nautical (-12°)
  - Astronomical (-18°)
- Interactive time controls:
  - Manual stepping (+10 min / -10 min)
  - Play / Pause animation
  - Live UTC mode
- Timeline showing:
  - Dawn / Sunrise / Sunset / Dusk
  - Overlap windows between stations
- “Best Dates” and “Best Window” analysis tools
- Fully client-side (no server required)
